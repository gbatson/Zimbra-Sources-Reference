User Migration Source Information
You can migrate data from an Outlook profile or a PST file.
Read the Migrating an Individual User Account to Zimbra Collaboration Server from Microsoft Exchange guide for complete migration details.
1.
Select to migrate an Outlook Profile or a PST File.
•
If Outlook Profile is selected, select the profile name from the menu and everything in the profile is migrated.
•
If PST File is selected, the entire .pst file is migrated.
2.
Click Next.
Use Load or Save
You can use Load or Save to save or upload profile information:
Click Load to upload previously saved profile information to populate the profile fields.
Click Save to save the profile information you are entering. This allows you to use this information at a later time
